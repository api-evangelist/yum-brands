## Note on API Availability

Yum! Brands is a restaurant operating and holding company. It does **not** publish a public developer API, developer portal, or OpenAPI documentation, and it maintains no public GitHub organization. Its technology is delivered internally through the proprietary **Byte by Yum!** SaaS platform and is gated to franchise, supplier, and partner integrations rather than a self-serve developer surface. This repository therefore profiles the company, its brands, and its technology footprint rather than a consumable API. No OpenAPI, AsyncAPI, JSON Schema, examples, or capability artifacts are generated because there is no real public API contract to back them.

## Byte by Yum! Technology Platform

Yum! Brands launched **Byte by Yum!** in 2025 — a proprietary AI-driven SaaS platform powering online and mobile-app ordering, point-of-sale, kitchen and delivery optimization, menu management, inventory management, labor management, and team member tools across all four brands. The platform is built in part on technology from companies Yum! has acquired (Dragontail Systems, Tictuk, Kvantum, QuikOrder) and on a partnership with NVIDIA for AI voice ordering and computer-vision drive-thru analytics.

## APIs

### Byte by Yum! Platform

The Byte by Yum! platform powers digital experiences across all Yum! Brands restaurants. While no public REST API is available, the platform includes:

- Digital ordering and mobile apps
- Point-of-sale systems
- Kitchen operations and display systems
- Menu management
- Inventory and labor management
- Customer loyalty programs
